# Virtual Pet App

## Overview
The Virtual Pet App is a fun and interactive application that allows users to take care of a virtual pet. Users can feed and pet their pet to keep it happy and healthy. The app provides a simple interface to manage the pet's state and view its information.

## Features
- **Petting the Pet**: Users can pet their virtual pet to increase its happiness.
- **Feeding the Pet**: Users can feed their pet to restore its health.
- **Pet Information**: View the pet's name, health, and happiness levels.

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm (version 6 or higher)

### Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd virtual-pet-app
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the App
To start the application, run:
```
npm start
```
This will launch the app in your default web browser.

### Running Tests
To run the unit tests, use:
```
npm test
```

## Project Structure
- `src/`: Contains the source code for the application.
- `public/`: Contains the static files, including the main HTML file.
- `package.json`: Lists the project dependencies and scripts.
- `tsconfig.json`: TypeScript configuration file.
